## Installation

### Easy step
* Open the extensions sidebar on Visual Studio Code
* Search for Cherry Wild Theme
* Click Install

### or, GitHub clone
```bash
cd ~/.vscode/extensions
git clone --depth=1 https://github.com/yyynnn/cherry-wild-vscode
```
